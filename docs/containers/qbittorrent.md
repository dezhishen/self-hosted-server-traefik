# qbittorrent 配置

| 设置名 | 配置项 | 说明 |
|--------|--------|------|
| 下载/默认保存路径 | `/data/downloads` | 其他分类目录务必设置在 `/data/` 下 |
| 下载/保存未完成的 torrent 到 | `/incomplete-torrents` | 未完成种子目录 |
| 下载/保存完成的 torrent 到 | `/finished-torrents` | 完成种子目录 |
| 连接/监听端口/UPnP | `true` | 自动映射端口 |
| BitTorrent/隐私/加密模式 | 强制加密 | 防止 ISP 限速 |
