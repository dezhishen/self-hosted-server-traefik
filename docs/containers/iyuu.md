# iyuu 配置

## 数据库配置

> 如果使用外部数据库才需配置，**在安装界面进行配置**

| 配置项 | 配置值 | 说明 |
|--------|--------|------|
| 数据库用户 | `root` | 可自行在 MariaDB 中创建 |
| 数据库密码 | `~/.args/MYSQL_PASSWORD` | — |
| 数据库 | `iyuu` | 必须手动创建 |
| 数据库 HOST | `mariadb` | 数据库容器名 |
| 数据库端口 | `3306` | — |

## qbittorrent 配置

| 配置项 | 配置值 | 说明 |
|--------|--------|------|
| 协议主机 | `http://${宿主机ip}:8080` | 需用宿主机 IP 或 traefik 网关 |
| 种子文件夹 | `/qbittorrent/config/qBittorrent/BT_backup` | — |
| 默认下载器 | `true` | — |

> 网关获取: `docker network inspect traefik --format '{{range .IPAM.Config}}{{.Gateway}}{{end}}'`

## transmission 配置

| 配置项 | 配置值 | 说明 |
|--------|--------|------|
| 协议主机 | `http://${宿主机ip}:9091` | 需用宿主机 IP 或 traefik 网关 |
| 接入点 | `/transmission/rpc` | — |
| 种子文件夹 | `/transmission/config/torrents` | — |
| 校验后做种 | `true` | — |
| 默认下载器 | **`false`** | 不做为默认下载器 |

## 自动转移任务

### 创建任务

1. 计划任务 → 任务管理
2. 点击「自动转移」
3. 添加任务

### 任务配置

| 配置项 | 配置值 | 说明 |
|--------|--------|------|
| 任务标题 | `qb-tr-自动转移` | — |
| 执行周期 | N 小时 | — |
| 来源下载器 | qbittorrent | 主下载器 |
| 目标下载器 | transmission | 辅下载器 |
| 标记规则 | 标记标签 | — |
| 路径转换类型 | 相等 | 已自动完成路径转换 |
| 跳校验 | NO | — |
| 暂停 | NO | — |
| 删除源做种 | **NO** | ⚠️ 首次转移勿勾选 |
| 启用 | `true` | — |

## 自动辅种任务

### 未开启自动转移时

| 配置项 | 配置值 |
|--------|--------|
| 任务标题 | `qb-tr-自动辅种` |
| 辅种站点 | 全选 |
| 辅种下载器 | qbittorrent |
| 主辅分离 | transmission |
| 通知渠道 | 爱语飞飞 |
| 自动校验 | On |

### 开启自动转移时

| 配置项 | 配置值 |
|--------|--------|
| 任务标题 | `tr-tr-自动辅种` |
| 辅种站点 | 全选 |
| 辅种下载器 | transmission |
| 主辅分离 | transmission |
| 通知渠道 | 爱语飞飞 |
| 自动校验 | On |
